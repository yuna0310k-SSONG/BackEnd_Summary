# 총정리

1. 라우팅 개념[crud<->get,post,delete,put]
2. get - params,query
3. post, delete, put - body
4. prisma
   - 1. npm install prisma: 프리즈마 다운
   - 2. npx prisma init: 프리즈마 할거임
   - 1] config.ts 지움
   - 2] 스키마 프리즈마에서 환경설정
   - 3. npx prisma db pull: db에 스키마를 가져옴
   - 4. npx prisma generate: 스키마를 자바스크립트 코드로 바꿈
   - 5. npm install @prisma/client@6: express랑 prisma 연결해주는 라이브러리
